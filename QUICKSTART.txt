========================================
TENG ML PLATFORM - GITHUB PAGES DEPLOYMENT
Quick Start Guide (5 Minutes)
========================================

STEP 1: CREATE GITHUB REPOSITORY
---------------------------------
1. Go to: https://github.com/new
2. Repository name: teng-ml-platform
3. Description: TENG Machine Learning Platform - Predict TENG Performance
4. Set to: PUBLIC (required for free GitHub Pages)
5. Do NOT check "Initialize with README"
6. Click "Create repository"

STEP 2: UPLOAD FILES
--------------------
Method A - Web Interface (Easiest):
1. On your new repository page, click "uploading an existing file"
2. Extract the ZIP file you received
3. Drag and drop these 5 files:
   - index.html
   - style.css
   - script.js
   - README.md
   - DEPLOYMENT_GUIDE.md
4. Commit message: "Initial commit - TENG ML Platform"
5. Click "Commit changes"

Method B - Command Line (If you have Git):
1. Extract ZIP file
2. cd teng_github_deploy
3. git init
4. git add .
5. git commit -m "Initial commit"
6. git remote add origin https://github.com/YOURUSERNAME/teng-ml-platform.git
7. git branch -M main
8. git push -u origin main

STEP 3: ENABLE GITHUB PAGES
---------------------------
1. Go to repository Settings
2. Click "Pages" in left sidebar
3. Under "Source":
   - Branch: main
   - Folder: / (root)
4. Click "Save"
5. Wait 1-2 minutes

STEP 4: ACCESS YOUR WEBSITE
---------------------------
Your website will be live at:
https://YOURUSERNAME.github.io/teng-ml-platform/

(Replace YOURUSERNAME with your actual GitHub username)
